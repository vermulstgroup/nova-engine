# `nova-engine pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
nova-engine pairing list whatsapp
nova-engine pairing approve whatsapp <code> --notify
```
